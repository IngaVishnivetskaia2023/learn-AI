### Guide Claude to relevant files

#### starting claude in project terminal

1.
```
claude
```

2. run init (Claude will analize the project and summarize in `claude.md`:
```
/init
```

3. check `CLAUDE.md` file content. This file helps Claude to better understand the codebase and this is a location where we can give Claude general guide.

`CLAUDE.md` is project level (shared with others).

`CLAUDE.local.md` has personal customizations (not shared).

Global `~/.claude/CLAUDE.md` used with all projects on the machine.

5. Activate in-memory mode with `#`, so you can edit `CLAUDE.md` file intelligenly.

6. Add context to conversation with `@`, like `@src/files/file.tsx`

7. If necessary add files to `claude.md` with `@`

8.
