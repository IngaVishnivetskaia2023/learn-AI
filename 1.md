### Machine Learning

#### Supervised Learning
Uses labeled data, where each input has a corresponding correct output, to train models to make predictions or classifications.  Commonly used for tasks like classification and regression.

#### Unsupervised Learning
Works with unlabeled data and aims to find hidden patterns, groupings, or structures within the data without explicit guidance on what the outputs should be. Used for clustering and dimensionality reduction.

#### Reinforcement learning (RLHF)
Reinforcement learning in conversational systems involves a structured methodology where AI agents 
evolve through ongoing user interactions. 
Central to this process is the iterative learning cycle, 
where each user interaction informs the AI's development. 
Positive interactions enhance the AI's strategies,
while negative ones prompt it to adjust and refine its approach.
